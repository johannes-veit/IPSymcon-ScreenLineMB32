# ScreenLine MB32 Controller

IP-Symcon Modul zur Steuerung einer ScreenLine MB32 Jalousie über LCN.

## Funktionen

- Auf / Ab Steuerung
- Positionsberechnung
- sichere Relaisverriegelung
- Fahrzeitüberwachung
- Shake-Free Unterstützung
- Anfängerfreundliche Konfiguration

## Installation

1. Repository in den IP-Symcon Module Ordner kopieren
2. Module neu laden
3. Instanz "ScreenLine MB32 Controller" erstellen

## Einrichtung

1. AUF Relais auswählen
2. AB Relais auswählen
3. Motorlaufzeiten eintragen
4. Testfahrt durchführen
5. Position prüfen

## Sicherheit

Das Modul verhindert:

- gleichzeitiges Einschalten von AUF und AB
- ungültige Relaiszuordnung
- unbegrenzte Fahrzeiten
